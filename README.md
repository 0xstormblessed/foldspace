## FoldSpace

**Onchain onboarding to Farcast and NFT wrapper for Farcast Id registration. Onboard to Farcast or gift a farcast id to someone else**

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```

### Gas Snapshots

```shell
$ forge snapshot
```
